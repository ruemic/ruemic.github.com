Upgrading the Blog

Sep 5th, 2012

Because if it ain’t fun, it won’t get done

I decided to start a blog earlier this year to journal my adventures in becoming a better coder/designer. I wanted my blog to be an example of my front end skills. So I was hand coding the html for each post. But it quickly become apparent that this was just way to much overhead to use, making posting feel like work instead of being fun, so it simply didn’t get done.

Lucky for me, some really smart folks have come up with a way to blog that really suits my purposes. Octopress. It’s a framework for Jekyll, a “blog aware static site generator” written in Ruby.
